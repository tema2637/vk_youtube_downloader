# YouTube/VK Media Downloader Bot

Этот Telegram-бот позволяет скачивать аудио и видео с YouTube и ВКонтакте.

## Установка

1. Установите зависимости:
   ```
   pip install -r requirements.txt
   ```

2. Добавьте токен вашего бота в файл `.env`:
   ```
   BOT_TOKEN=ваш_токен_бота
   ```

3. (Опционально) Установите FFmpeg для извлечения аудио:
   - Скачайте с https://ffmpeg.org/download.html
   - Добавьте путь к папке bin в переменную среды PATH
   - Или укажите путь к ffmpeg в файле `.env`:
     ```
     FFMPEG_PATH=путь_к_ffmpeg.exe
     ```

## Использование

1. Отправьте команду `/start`, чтобы получить инструкции
2. Отправьте боту ссылку на видео с YouTube или ВКонтакте, или используйте команду `/search` для поиска видео по названию
3. Бот предложит выбрать: "Аудио" или "Видео"
4. Нажмите на нужную кнопку для загрузки
5. Файл будет скачан с названием видео
